📌 解析规则说明（基于PC端waveform.html）：

  • 寄存器0 → 土壤湿度 (÷10) %
  • 寄存器1 → 土壤温度 (补码处理后÷10) ℃
  • 寄存器2 → 电导率 (直接显示) μS/cm
  • 寄存器3 → PH值 (÷10，不是÷100！)
  • 寄存器4 → 氮含量 mg/kg
  • 寄存器5 → 磷含量 mg/kg
  • 寄存器6 → 钾含量 mg/kg
  • 寄存器7 → 盐度 (÷10) ‰
  • 寄存器8 → TDS mg/L
